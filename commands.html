役職パネルbotV2(!rp2)

公式コマンド一覧

add

パネルを追加します。

  • 権限X(例)

    • 指定した権限をパネルに追加します。

  • --tag

    • パネルにタイトル（タグ）が設定される。（設定しないとデフォルトと言うタイトル（タグ）になる）

  • --channel

    • パネルを設置するチャンネルを指定する

  • --emoji

    • 権限付与する為のリアクション絵文字の指定をする。

  • --create

    • 役職を作ってからパネルを作る

  • --color

    • パネルの色を指定する

remove

パネルから役職を削除します。

  • 権限X(例)

    • その権限をパネルから削除します

  • --tag

    • 指定のタイトル（タグ）から役職を削除します

  • --channel

    • 削除するチャンネルをします

  • --delete

    • パネルから役職を削除した後にその役職も削除されます。

autoremove

パネル内のもう無い役職を削除します。

edit

パネルを編集します。

  • メッセージid

    • 編集するパネルのメッセージidを指定する

  • --tag

    • タイトル（タグ）の別のものに変更します

  • --color

    • パネルを別の色に変更します

  • --channel

    • 編集するパネルのチャンネルを指定します

role

ロール一覧をロールidと共に表示します

  • ページ2(例)

    • 役職一覧のページを指定する

find

パネルを検索してあった場合そのメッセージidを返します。

  • タグX(例)

    • 検索したいパネルのタイトル（パネル）を指定する

  • ページ

    • 検索したいパネルのページを指定する

  • --channel

    • 検索したいパネルのチャンネルを指定できる

    指定しなかった場合、コマンドを打ったチャンネルと同じになる

refresh

パネルのリアクションを付けなおす

  • メッセージid

    • リアクションを付けなおすパネルを指定する

  • --channel

    • リアクションを付けなおすチャンネルを指定する

sort

パネルのリアクションをアルファベット順に並べる事が出来る

  • メッセージid

    • アルファベット順に並べるパネルを指定する

  • --channel

    • アルファベット順に並べるチャンネルを指定する

copy

パネルを複製する事が出来る

  • メッセージid

    • 複製元のパネルを指定する

  • --from

    • 複製元のチャンネルを指定する

  • --channel

    • 複製先のチャンネルを指定する

reset

autoremove、sort、refreshを一気に行います。 以下の順番で行われます。 →存在しない役職の行を削除（autoremove) →アルファベットの絵文字はアルファベット順に編集（sort） →リアクションの付け直し（refresh）
  • メッセージid

    • それを行うパネルを指定する

  • --channel

    • それを行うチャンネルを指定する
